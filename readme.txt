Use Cinema - Machine Cinema and Scheduling System Plugin for cinema and managing cinema  in your WordPress based websites.

== Description ==

FastBook - Machine Cinema plugin facilitates a WordPress based website to accept cinema using a form which is controlled by a shortcode ([cinema]).
Admin just places the shortcode in any page or post and the cinema form appears in the front end from where users can book for machines.
All the submitted cinemas by the users are then visible to the admin in the admin area. 


Features:
 
* You can use this in any specific page/post by putting shortcode [cinema].

== Installation ==

1. Upload the Appointment folder to the /wp-content/plugins/ directory.
2. Activate the plugin using the 'Plugins' menu in your WordPress admin panel.
3. You can adjust the necessary settings using your WordPress admin panel in 
   "Cinema Settings ".
4. Create a page or a post and insert the shortcode [cinema] into the text.

== Frequently Asked Questions ==
Q.1 : Where do I check the booked appointments in the admin panel ?
Answer : In the admin panel, go to menu 'Cinema System' --> Cinema Lists

Q.2 : Along with the labels, Can I change the 'Success Message' also ?
Answer : Yes, you can change that too. In the admin panel, go to menu Cinema Settings and change your message easily.
